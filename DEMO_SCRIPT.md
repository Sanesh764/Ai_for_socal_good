# Campus Compass - 2-Minute Demo Script

## Introduction (15 seconds)
"Hi! I'm [Name], and I'm excited to present **Campus Compass** - an inclusive, accessibility-first platform that uses responsible AI to improve wellbeing and information access for all campus community members, especially those with disabilities."

## Problem Statement (20 seconds)
"Traditional campus platforms often lack accessibility features, making it difficult for visually impaired, dyslexic, and neurodiverse students to access support. During emergencies, clear communication is critical. Campus Compass addresses these challenges by combining responsible AI with human-centered design."

## Key Features Demo (60 seconds)

### 1. Accessibility Features (15 seconds)
"First, let me show you our accessibility-first design. [Click accessibility controls]
- High contrast mode for better visibility
- Dyslexia-friendly fonts and spacing
- Text-to-speech for visually impaired users
- Full keyboard navigation and screen reader support"

### 2. AI Wellbeing Companion (20 seconds)
"Next, our AI wellbeing companion powered by Gemini API. [Navigate to wellbeing page]
- Calm, supportive responses in English and Hindi
- Non-diagnostic, verified information only
- Automatic distress detection that escalates to human support
- Clear disclaimers that this is an assistant, not a replacement for professional help"

[Type a sample query: "I'm feeling stressed about exams"]
[Show response, highlight disclaimers]

### 3. Campus Alerts (15 seconds)
"Our campus alerts system provides verified, human-reviewed information. [Navigate to alerts]
- Admin-approved alerts for weather, safety, and emergencies
- Clear, accessible presentation
- Non-panic-inducing messaging"

### 4. AI Safety & Ethics (10 seconds)
"Finally, our safety layer includes input filtering, output moderation, and sensitive query logging. Distress situations automatically trigger crisis resources and human support guidance."

## Technical Highlights (15 seconds)
"Built with Node.js, Express, EJS, and Gemini API. Plain CSS for performance and compatibility. MongoDB for data storage. All designed to be deployable and maintainable."

## Impact & Future (10 seconds)
"Campus Compass demonstrates how responsible AI can be a force for social good. Future enhancements include additional languages, enhanced moderation, and analytics dashboards."

## Closing (5 seconds)
"Thank you! Campus Compass - making technology accessible and supportive for everyone. Questions?"

## Key Talking Points

1. **Accessibility First:** Not an afterthought, built from the ground up
2. **Responsible AI:** Safety measures, disclaimers, human escalation
3. **Social Impact:** Serves underserved communities (visually impaired, dyslexic, neurodiverse)
4. **Practical & Deployable:** Realistic for hackathon, can be production-ready
5. **Ethical Design:** Human-in-the-loop, no medical/legal advice, transparency

## Potential Questions & Answers

**Q: How do you ensure AI safety?**
A: Multiple layers - input validation, distress detection, output moderation, and sensitive query logging. All responses include disclaimers.

**Q: What about privacy?**
A: We only log queries containing distress indicators. No personal information is collected. All data is securely stored.

**Q: Can this replace human counselors?**
A: Absolutely not. The AI is an assistant that provides general information and escalates serious concerns to human support immediately.

**Q: How accessible is it really?**
A: We meet WCAG 2.1 Level AA standards, with many features exceeding to Level AAA. Tested with screen readers and assistive technologies.

**Q: What's the tech stack?**
A: Node.js, Express, EJS for templating, Gemini API for AI, MongoDB for storage, plain CSS for styling. Simple, maintainable, and performant.
